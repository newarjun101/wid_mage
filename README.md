# Widmage

A flutter library to convert Widget to Image.

## Installation

Go to the __*pubspec.yaml*__ to install foobar.

```bash
dependencies:
  wid_mage : any
```

### Import

```Dart
import 'package:wid_mage/wid_mage.dart';

```
